novice
======

Some assembly tools

Getting Started
---------------

    $ npm install --dev
    $ npx lerna bootstrap
    $ npm run build
    $ ./novice --help

Useful stuff:

    $ npm run lint
    $ npm run lint:fix
    $ npm test
    $ npm test:cover
    $ ./bootstrap-parse-table.sh complx
    $ ./generate-parse-table.sh complx
